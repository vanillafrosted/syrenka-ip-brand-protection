# Editorial Checklist

**Purpose**: Verification checklist for each article before publication.

Use this checklist to ensure every article meets English Syrenka standards.

---

## Pre-Writing Checklist

- [ ] Source material reviewed and accessible
- [ ] Japanese source article read and understood
- [ ] Ray's World Library reviewed for appropriate details
- [ ] Scene outline approved by editor
- [ ] Fact-check points identified

---

## During Writing Checklist

- [ ] Following Editorial Manifesto principles
- [ ] Using Scene Bible techniques
- [ ] Present tense, immediate moment maintained
- [ ] Reader walks beside Ray (not as Ray)
- [ ] Ordinary moments protected
- [ ] Small details included
- [ ] Silence and space respected
- [ ] Ray remains distinct character
- [ ] Tokyo integrated naturally
- [ ] No explanation of emotions

---

## Post-Writing Checklist

- [ ] All scenes reviewed by human editor
- [ ] Photo placeholders positioned correctly
- [ ] Factual accuracy verified against source
- [ ] Ray's canonical facts honored
- [ ] Timeline consistent with article series
- [ ] Tone consistent with Manifesto
- [ ] No invented experiences
- [ ] Privacy boundaries respected

---

## Before Publication Checklist

- [ ] Final fact-check complete
- [ ] SEO elements optimized
- [ ] Mobile layout tested
- [ ] Links verified
- [ ] All sections present and in order
- [ ] Draft banner removed
- [ ] Meta tags set correctly
- [ ] Ready for live deployment
