## Workflow for Each Article

### Step 1: Copy the Master

```
Source: /assets/syrenka/master/syrenka_master_v2.png
Copy to working directory (temporary)
```

**Do NOT edit the master template directly. Always work on a copy.**

### Step 2: Add English Text Only

You will receive:
- Article number (e.g., "Article No.3")
- Approved English text for the speech bubble (1-2 sentences)
- Text specifications:
  - Font: Helvetica Regular, 42 px
  - Color: RGB(40, 40, 40)
  - Line spacing: 1.28
  - Left alignment
  - Position: X=735px, Y=356px (approximately)
  - Manual line breaks only

### Step 3: Text Fitting Rules (Priority Order)

1. **Shorten text if it doesn't fit** (preferred solution)
2. Never reduce font size
3. Never modify layout
4. Never move text position
5. Never redesign template

**Layout integrity has absolute priority over text length.**

If text doesn't fit at 42px in the allocated space:
- Work with Marcus/Maki to shorten the text
- Do NOT reduce font size
- Do NOT move the speech bubble

### Step 4: Save as Article-Specific Image

**Filename format:** `articleXXX-syrenka-speech-bubble-final.png`

**Location:** `/public/assets/images/`

**Example:**
- Article No.3: `article003-syrenka-speech-bubble-final.png`
- Article No.5: `article005-syrenka-speech-bubble-final.png`

### Step 5: Verification

Before submitting, verify:
- ✓ All text is fully visible inside the speech bubble
- ✓ No text extends beyond right border
- ✓ Comfortable blank space remains around text
- ✓ Text does not extend above or below allocated area
- ✓ Syrenka character is unchanged
- ✓ Speech bubble design is unchanged
- ✓ Canvas is still 1661 × 947 px
- ✓ File is saved as PNG with no transparency
