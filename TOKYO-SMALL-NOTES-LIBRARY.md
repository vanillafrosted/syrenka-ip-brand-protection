# Tokyo Small Notes Library

**Effective Date**: July 8, 2026

**Purpose**: A shared editorial library containing authentic moments from Ray's real life while building and defending a small brand in Tokyo.

This is NOT a culture guide. This is NOT a travel guide.

This is a collection of genuine experiences that help international readers naturally experience Tokyo through Ray.

---

## How to Use This Library

Before writing each article, review this library.

Christopher should choose only the notes that naturally fit the story.

Never force a Tokyo Small Note into an article.

Each note should enhance the narrative and help readers feel they are walking beside Ray in Tokyo.

---

## Library Structure

Each entry includes:

- **ID**: Unique identifier (TSN-001, TSN-002, etc.)
- **Title**: The authentic moment
- **Category**: Type of experience (Court, Daily Business, Transportation, Office, etc.)
- **Description**: What actually happened
- **Suggested Article Themes**: Articles where this note might naturally fit
- **Used in Article**: Track which articles use this note (populated over time)
- **Photo Opportunity**: Yes/No - Can this moment be captured in a photograph?
